----------------------
SETUP:
----------------------
1.) Place files of you local squirrel 3.7.1 setup dir into ".\Squirrel-SQLPortable\App\squirrel-sql-3.7.1-standard"
3.) Start launcher with ".\Squirrel-SQLPortable\Squirrel-SQLPortable.exe"



----------------------
Additional Requirements:
----------------------
- local installed Java on machine or Java from PortableApps
- place your JDBC driver jars at ".\Squirrel-SQLPortable\Data\db-drivers\" to import them automatically during start
